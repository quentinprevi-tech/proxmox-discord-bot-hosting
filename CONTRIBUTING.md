# Contributing

This repository is mainly a portfolio and documentation version of a homelab project.

Contributions are not actively expected, but suggestions, corrections or feedback are welcome.

## Scope

This repository contains:

- project documentation
- architecture overview
- monitoring documentation
- security notes
- sanitized example scripts
- screenshots with sensitive information removed

It does not contain production secrets, private configuration files, customer data or internal backups.

## Security

Do not submit:

- Discord bot tokens
- Stripe API keys
- webhook secrets
- admin passwords
- private IP addresses
- internal URLs
- production databases
- customer data

## Suggestions

Useful suggestions may include:

- documentation improvements
- security recommendations
- clearer architecture explanations
- better example scripts
- monitoring improvements

## Project status

This project is a homelab / private beta portfolio project.

It is not presented as a production-ready hosting platform with guaranteed availability.
