# Roadmap

This roadmap lists possible improvements for the platform.

## Short Term

- Add a customer guide explaining how to create a Discord bot and retrieve a bot token.
- Improve the public order tracking page with clearer user instructions.
- Add more screenshots to the GitHub repository.
- Add sanitized script examples.
- Improve deployment documentation.

## Medium Term

- Add email notifications for order status changes.
- Add better subscription status handling.
- Add upgrade and downgrade support between plans.
- Improve logging and error visibility in the admin dashboard.
- Add stronger Discord token storage protection.

## Long Term

- Add per-customer authentication.
- Add customer-facing metrics dashboards.
- Add automated backups for customer bot data.
- Add stronger isolation and resource enforcement.
- Add a more production-ready deployment model.

## Notes

This project is currently a homelab/private beta project.

The main goal is to demonstrate infrastructure automation, monitoring, payment lifecycle handling and Linux/Proxmox administration skills.
